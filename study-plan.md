## Critical exam traps

1. **MSL vs AGL** — sectional chart altitudes are MSL; Part 107 altitude limit is AGL.
2. **Below 400 ft ≠ legal** — controlled airspace still requires authorization.
3. **Gust spread** — gusts exceeding aircraft wind rating = no-go even if sustained wind is OK.
4. **Visibility on charts** — station plots use SM; minimum Part 107 = 3 SM.

## Suggested drill order

1. Figure 26 — runway relative position (left downwind RWY 13)
2. Figure 21 — MOA active status (Military Operations Directory)
3. Figure 59 — VR routes altitude limits
4. Full 20-question timed mock using only supplement figures
