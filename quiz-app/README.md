# Part 107 Quiz Lab

Study app for the FAA Part 107 / UAG knowledge test.

## Run locally

```bash
./start.sh
```

Opens http://127.0.0.1:8765

On the live site: open `/quiz-app/`.

## Features

- Inline FAA testing-supplement figures
- Every question includes **D. I don't know** → shows answer + explanation
- Decks: charts, mixed, regs-only, RWY 13 workshop
- Progress in `localStorage`
